Image cipher

This app works as a image cryptography and steganography tool.

  • Cryptography: It makes your image to pure noise by encrypting it with a secret key.
  • Steganography: It hides a secret message inside an image.
The code runs entirely in your browser, so your data is always safe and private :)

Mode: Cryptography Steganography
Select an image file and enter a secret to encode / decode.

Drag and drop an image file here.

Supported formats: PNG, JPEG, BMP, ...
